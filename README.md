# API para gerenciamento de vídeos do Youtube em JSON!

## Para que esta API foi feita?

Esta API foi desenvolvida com o objetivo de armazenar e organizar links de aulas do YouTube, facilitando o acesso aos seus conteúdos de estudo.
Embora simples, ela foi criada para tornar sua rotina mais prática, permitindo categorizar vídeos por tipo de aula e tema abordado.
